# 1.2.4 求幂
## 练习1.18
利用练习1.16和练习1.17的结果设计一个函数，它能产生一个基于加、加倍和折半运算的迭代计算过程，只用对数的步数就能求出两个整数的乘积。

## 解答
```rust
fn fast_times(mut b: i64, mut n: i64) -> i64 {
    let mut a = 0;
    while n != 0 {
        // 不断迭代b的累加和，多出来的1，就先加到a上，直至n=0
        if n % 2 == 1 {
            a = a + b
        }
        n = n / 2;
        b = b * 2;
    }
    a
}
```
